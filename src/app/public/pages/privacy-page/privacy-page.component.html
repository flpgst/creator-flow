Política de privacidade

Esta página descreve, em linguagem simples, quais dados o Creator Flow usa para conectar o YouTube, importar comentarios e apoiar a criação de roteiros.

Dados tratados

O app pode tratar dados de conta, identificadores do canal do YouTube, vídeos, comentarios importados, selecões feitas pelo usuário e roteiros criados dentro do Creator Flow.

Finalidade

Os dados são usados para autenticar o usuário, manter a conexão com o YouTube, importar comentários autorizados e permitir que o usuário organize esses comentários em roteiros.

Controle do usuário

O usuário pode revogar o acesso no Google, deixar de usar a integração e solicitar remoção de dados conforme os canais de contato publicados pelo serviço.
